# Output Tables and Figures

This folder is reserved for derived tables and figures generated by the RMCT reproducibility workflow.

## Expected output categories

- `rmct_searchable_vocabulary.csv`: one row per searchable preferred or alternative term.
- `rmct_company_profile_matches.csv`: company-by-capability match rows generated from authorised local website text.
- `benchmark_retrieval_summary.xlsx`: benchmark results for supplier-search candidate-set reduction.
- `expert_validation_metrics.xlsx`: agreement and precision metrics after two expert reviewers complete the validation template.

## Data-sharing note

Generated outputs should not include full raw or cleaned website text. Where evidence is needed, share short extracted spans, document hashes, aggregate counts, anonymised company identifiers and RMCT labels.

## Recommended workflow

1. Run preprocessing and hashing locally if full website text is available.
2. Build or load the RMCT searchable vocabulary.
3. Create company-level profiles from searchable term matches.
4. Run benchmark calculations.
5. Add completed expert ratings and calculate agreement metrics.

The public repository contains reproducibility-oriented data and code, while full website texts remain excluded for copyright and research-governance reasons.
